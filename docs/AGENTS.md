# Docs Agent Notes

This tree is the canonical home for durable repository guidance.

- Run the `docs-sync` skill when a change affects durable behavior, contracts, workflow, testing policy, or architecture rules.
- Keep one canonical home for each durable rule. Prefer links over repetition.
- Update the matching category index when adding or moving a permanent doc.
- If code and docs disagree, fix the stale doc in the same change.
